# ScriptableObjects


## Download
1. Download the [Standalone](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/GameDevPyro/ScriptableObjects/tree/master/Standalone).
2. Unzip the .zip document.
3. Run "Scriptable Objects.exe"
